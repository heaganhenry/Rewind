									let userString = document.getElementById("userString").value;
									let reversedString = reverseString(userString);

								// logic function
								// reverse the string
								function reverseString(userString) {
									let reversedString = "";
									// reverse the string using a for loop
									for (let i = userString.length - 1; i >= 0; i--){
										reversedString += userString[i];
									}
								
									return reversedString;
								}
								
								// view function
								// display the reversed string
								function displayString(reversedString) {
									// insert string in the page
									document.getElementById("msg").innerHTML = `Your string reversed is: ${reversedString}`
								
									// show the alert box
									document.getElementById("alert").classList.remove("invisible");
								}

The Code is structured in three functions.

getValue

getValue is a function that retrieves the value of the string entered by the user on the web page. This is achieved by using the native getElementById function and referencing the unique id of the input field used in the web page. Being a controller function getValue also calls the below functions to control when they are run.

reverseString

reverseString is a function that takes the user input text and reverses it by creating a new empty string and inserts each character of the user text into the new string, starting from the last character to the first.

displayString

displayString is a function that takes the reversed string created by the previous function and inserts it into the web page by referencing a unique id at the appropriate position on the page with the native innerHTML function.
